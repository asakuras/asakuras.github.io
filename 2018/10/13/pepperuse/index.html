Pepper使用心得

发表于2018-10-13|更新于2021-04-26|study
|字数总计:208|阅读时长:1分钟|阅读量:

  • 通过ssh连接pepper头部linux系统后，使用qicli call ALTabletService._launchApk com.android.settings可进入平板设置界面

  • 使用 qicli call ALTabletService.showWebview your_url 可直接让平板访问网页
    机器人出厂状态下就有的几个机器人应用程序如下：

    • ALTactileGesture`
    • Animation Library
    • Animation mode
    • boot-config
    • j-tablet-browser
    • robots-advanced
    • signs-and-feedback
    • Sleep Reactions

  • .top 对话文件应该被放置在naoqi的/home/nao/.local/share/PackageManager/apps/目录下

  • j-tablet-browser 用于通过电脑或者pepper端直接显示平板的内在界面

  • 调用科大讯飞的接口的方法就是在freechat的的listen class里使用listen的字符串
    注意事项：使用choregraphe时，路径中不能出现中文，否则会报错
    使用freechat的main.py时需要修改命令行的字符编码, mode con cp select=65001,修改成utf-8格式
